# Связывание набора данных с сервисом {#linking-dataset-or-service}

Метаданные набора данных могут быть связаны с метаданными сервисов.

В записи метаданных сервиса нажмите кнопку `Связать с набором данных`, чтобы открыть cписок записей, где перечислены все наборы данных в каталоге. Имя слоя может быть задано вручную или из списка слоев, полученных из сервиса (с помощью GetCapabilities).

![](img/dataset.png)

В записи метаданных набора данных нажмите кнопку `Связать с сервисом`, чтобы открыть cписок записей, где перечислены все сервисы в каталоге.

![](img/service.png)

При подтверждении связи изменения затронут обе записи (в зависимости от привилегий пользователя - т. е. текущий пользователь может не иметь права изменять целевую запись).
